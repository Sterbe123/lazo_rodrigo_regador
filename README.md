# lazo_rodrigo_regador

Supón que fuiste contratado(a) como ingeniero(a) trainee en una empresa tecnológica. Dentro de
tus actividades debes asistir a una reunión de toma de requerimientos. El cliente requiere de una
aplicación móvil para controlar y monitorear sistemas de regadío remoto que se encuentran a lo
largo del país, y debe contar con login que valide usuario y contraseña. Tú, junto al equipo que asiste
a la reunión, proponen que la aplicación será desarrollada en lenguaje Kotlin y se hará entrega de un
aplicativo demo (que mostrará su nombre – apellido – sección del curso - fecha).
